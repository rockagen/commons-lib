##commons-lib

commons lib for java,some wrapper functions .


## Usage
`maven:`
```java
<dependency>
	<groupId>com.rockagen</groupId>
	<artifactId>commons-lib</artifactId>
	<version>0.1.10-RELEASE</version>
</dependency>
```

`gradle:`
```groovy
'com.rockagen:commons-lib:0.1.10-RELEASE'
```
